HEMATOPOIETIC STEM CELL (HSC)   [CD34+, multipotent, self-renewing]
│
├──▶ COMMON MYELOID PROGENITOR (CFU–GEMM)
│       ("Granulocyte, Erythrocyte, Monocyte, Megakaryocyte")
│
│   ├──▶ ERYTHROID LINEAGE
│   │       ├── BFU–E (Burst-Forming Unit – Erythroid)
│   │       ├── CFU–E (Colony-Forming Unit – Erythroid)
│   │       ├── Proerythroblast
│   │       ├── Basophilic Erythroblast
│   │       ├── Polychromatophilic Erythroblast
│   │       ├── Orthochromatic (Normo-) Erythroblast
│   │       ├── Reticulocyte  →  **ERYTHROCYTE (RBC)**
│   │       │
│   │       └── Stimulated by: **EPO (Erythropoietin, kidney)**
│   │
│   ├──▶ GRANULOCYTIC LINEAGE
│   │       ├── CFU–GM  →  Myeloblast
│   │       │     ├── Promyelocyte
│   │       │     ├── Myelocyte
│   │       │     ├── Metamyelocyte
│   │       │     ├── Band form
│   │       │     └── **Neutrophil (CFU–G)** or **Monocyte (CFU–M) → Macrophage**
│   │       ├── CFU–Eo  →  **Eosinophil**
│   │       └── CFU–Ba  →  **Basophil**
│   │
│   │       Stimulated by:
│   │       • **G–CSF** → Neutrophils
│   │       • **M–CSF** → Monocytes
│   │       • **IL–5**  → Eosinophils
│   │       • **IL–3, GM–CSF** → General myeloid support
│   │
│   ├──▶ MEGAKARYOCYTIC LINEAGE
│   │       ├── CFU–Meg  →  Megakaryoblast
│   │       ├── Megakaryocyte (polyploid nucleus)
│   │       └── **Platelets (Thrombocytes)**
│   │
│   │       Stimulated by: **TPO (Thrombopoietin, liver/kidney)**
│   │
│   └──▶ MONOCYTIC LINEAGE
│           ├── CFU–M  →  Monoblast → Promonocyte → **Monocyte → Macrophage**
│           └── Cytokines: **M–CSF, GM–CSF**
│
└──▶ COMMON LYMPHOID PROGENITOR (CFU–L)
        ("Lymphoid stem cell")
        │
        ├──▶ B–CELL LINEAGE
        │       ├── Pro–B cell → Pre–B cell → Mature B cell
        │       └── Plasma cell (antibody-secreting)
        │
        ├──▶ T–CELL LINEAGE
        │       ├── Pro–T cell → Thymic maturation
        │       ├── CD4+ Helper T cells
        │       └── CD8+ Cytotoxic T cells
        │
        └──▶ NK–CELL LINEAGE
                └── **Natural Killer Cell**
                │
                Stimulated by: **IL–7 (critical for lymphoid development)**
